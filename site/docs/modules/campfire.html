  • insert: ((elem: Element, where: ElementPosition) => Element)
      • (elem: Element, where: ElementPosition): Element

      • Inserts an element into the DOM given a reference element and the relative position of the new element.

        • if where looks like { after: reference }, the element is inserted into reference's parent, after reference.
        • if where looks like { before: reference }, the element is inserted into reference's parent, before reference.
        • if where looks like { prependTo: reference }, the element is inserted into reference, before its first child.
        • if where looks like { appendTo: reference }, the element is inserted into reference, after its last child.
        throws

        an Error when there are either zero or more than one keys present in where.

        Parameters

        • elem: Element

          The element to insert.

        • where: ElementPosition

          An object specifying where to insert elem relative to another element.

        Returns Element

        the element that was inserted, so you can do const a = insert(nu(), _).

  • mustache: ((string: string, data?: Record<string, string>, shouldEscape?: boolean) => string)
      • (string: string, data?: Record<string, string>, shouldEscape?: boolean): string

      • Applies mustache templating to a string. Any names surrounded by {{ }} will be considered for templating: if the name is present as a property in data, the mustache'd expression will be replaced with the value of the property in data. Prefixing the opening {{ with double backslashes will escape the expression. By default, mustache data is escaped with campfire's escape() function - you can disable this by supplying the value of esc as false.

        Parameters

        • string: string

          the string to be templated.

        • data: Record<string, string> = {}

          The data which will be used to perform replacements.

        • shouldEscape: boolean = true

          Whether or not the templating data should be escaped. Defaults to true.

        Returns string

        the templated string.

  • nu: ((eltInfo: string, args?: ElementProperties) => (null | HTMLElement)[])
      • (eltInfo: string, args?: ElementProperties): (null | HTMLElement)[]

      • An element creation helper.

        example
        cf.nu(`elt#id.class1`, {
         raw: true,
         c: html`<span class=some-span>foo bar</span>`,
         gimme: ['.some-span']
        }) // Output: [<elt#id.class1>, <the span some-span>]
        example
        cf.nu(`span.some-span`, {
         // properties...
         // no gimme specified
        }) // Output is still a list [<span.some-span>]

        Parameters

        • eltInfo: string

          Basic information about the element. eltInfo should be a string of the format tagName#id.class1.class2. Each part (tag name, id, classes) is optional, and an infinite number of classes is allowed. When eltInfo is an empty string, the tag name is assumed to be div.

        • args: ElementProperties = {}

          Optional extra properties for the created element.

        Returns (null | HTMLElement)[]

        The newly created DOM element and any other elements requested in the gimme parameter specified in args.
