    Class ListStore<T>

    A reactive list store. Implements push(item). remove(idx), get(idx), and setAt(idx, item). push() sends an "append" event remove() sends a "deletion" event setAt() sends a "change" event with idx

    on

    • on(
          type: "clear" | "change" | "append" | "deletion",
          fn: Subscriber,
          callNow?: boolean,
      ): number

      Add an event listener to the store.

      Parameters

      • type: "clear" | "change" | "append" | "deletion"

        The type of event to listen for. Supported event types include:

        • 'change': Triggered when the store's value is updated via update().
        • 'append': For ListStore - Triggered when an item is added to the list.
        • 'deletion': For ListStore/MapStore - Triggered when an item is removed.
        • 'clear': Triggered when the store is cleared.
      • fn: Subscriber

        A callback function that will be invoked when the specified event occurs. The function receives a StoreEvent object with details about the event.

      • callNow: boolean = false

        Determines whether the callback should be immediately invoked with the current store value. Only applies to 'change' event type.

      Returns number

      A unique subscriber ID that can be used to unsubscribe the listener.
